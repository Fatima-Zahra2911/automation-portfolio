Face Mask Detection System
End-to-end computer vision pipeline for real-time face mask detection using Haar Cascade face detection, ROI preprocessing, and classical ML classifiers. Deployed as a live web app with Flask.
Best Accuracy 85.7%
Models Used Logistic Reg + SVM
Tuning Method GridSearchCV

What is this doing? The system uses a pre-trained Haar Cascade detector (a fast face-finding algorithm) to locate faces in images. It then crops just the face region (ROI) and feeds it to a classifier.

The ML part: Two classifiers were trained — Logistic Regression (a simple linear model) and SVM (Support Vector Machine, which finds the best boundary between mask/no-mask). GridSearchCV automatically tested many combinations of settings to find the best ones.

Deployment: Wrapped in a Flask web app so anyone can upload a photo and get a prediction in their browser.

Beginner–Intermediate
E-Commerce Jewelry Sales Predictor
Regression model that predicts the order value of jewelry purchases from transaction-level data. Uses advanced feature engineering and a Random Forest Regressor, deployed via Flask.
RMSE Score ≈ 2.15
Model Random Forest Regressor
Task Type Regression

What is this doing? Given information about a transaction (items, category, time of purchase, customer data), the model predicts how much the order will be worth — like predicting the bill before checkout.

The ML part: A Random Forest is an ensemble of decision trees — each tree makes its own guess, and the forest averages them out. This makes it robust against noise. Feature engineering means creating new smart columns from raw data (e.g. "weekend purchase", "avg past spend") to help the model learn better patterns.

RMSE of 2.15: The model's predictions are off by only ~2.15 units on average — a solid result for this kind of data.
